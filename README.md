# Booking Mobile App

Mobilapplikation byggd med React Native och Expo.

## Köra Mobilappen

### 1. Klona projektet

git clone https://github.com/AmerAlzor/booking-app.git
cd booking-app

---

### 2. Installera beroenden

npm install

eller

yarn install

---

### 3. Skapa miljövariabler

Skapa en `.env`-fil i projektets root och lägg in följande:

EXPO_PUBLIC_API_BASE_URL=https://platform-admin-sepia.vercel.app
EXPO_PUBLIC_API_URL=https://platform-admin-sepia.vercel.app

---

### 4. Installera Expo Go

Ladda ner **Expo Go** från App Store eller Google Play.

---

### 5. Starta appen

npx expo start

eller rensa cache:

npx expo start -c

---

### 6. Starta appen på en enhet

När Expo startar kan du:

Tryck i → Starta iOS simulator
Tryck a → Starta Android emulator

Eller:

- Skanna QR-koden med Expo Go-appen på din mobil.

---

## Krav

- Node.js
- Expo Go app
- iOS Simulator eller Android Emulator (valfritt)
